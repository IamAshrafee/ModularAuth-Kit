# AI-Assisted Integration

Use an AI coding agent (Cursor, GitHub Copilot, Gemini, etc.) to integrate ModularAuth-Kit into your project with **a single prompt**.

## How It Works

1. Copy the `src/auth/` folder into your project
2. Give the AI agent the prompt from [agent-prompt.md](agent-prompt.md)
3. The AI asks you a few questions → configures everything → done

## Files

| File | Purpose |
|---|---|
| [agent-prompt.md](agent-prompt.md) | The main prompt to give your AI agent |
| [context.md](context.md) | Technical context the AI needs to understand the module |
